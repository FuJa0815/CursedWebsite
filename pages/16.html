CREATE OR REPLACE PROCEDURE in_out_proc (num IN OUT NUMBER) IS BEGIN num := num + 1; END; / DECLARE num NUMBER := 50; BEGIN in_out_proc(num); DBMS_OUTPUT.PUT_LINE(num); END; / DROP PROCEDURE in_out_proc;

IN OUT parameters are readable and writeable

Using IN OUT parameters the caller can provide a value to the procedure and the procedure can override the value.

Often times stored functions would make more sense.

IN OUT parameters are passed by value.
